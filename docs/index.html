rubygana

漢字かんじにルビ(ふりがな)を付つけちゃうぞ。

  • HTML
  • テキスト文書ぶんしょ
  • マークダウン

ルビ族のルビ族によるルビ族のためのコマンドルビ族女帝にあたいはなるっ!

  • 漢字かんじを学まなび始はじめた人ひと
  • 教育きょういく関係かんけい者しゃ、出版しゅっぱん社しゃ
  • ルビ付つきHTMLを迫せまられて悩なやむエンジニア

ルビ族ぞく以外いがいもどうぞ!

特徴とくちょう(rubygana 0.8.1)

  1. HTMLのルビ化か

    <ruby><rb>蝙蝠</rb><rt>こうもり</rt></ruby>

  2. テキスト文書ぶんしょのルビ化か

    蝙蝠(こうもり)

  3. ルビ化かずみHTMLに学年がくねんクラス名めい挿入そうにゅう

    <ruby class="学年3 学年1"><rb>落下</rb><rt>らっか</rt></ruby>

  4. マークダウンのHTML5化か
  5. テキスト文書ぶんしょのHTML5化か

ルビの調整ちょうせい

  • ルビ指定してい: 自動じどうのルビが不味いん?! マジぃときは指定してい
    猫の一撃(ネコパーンチ)

  • 学年がくねん指定してい: 自動じどうルビ化かの範囲はんいを小学しょうがく3年ねん以降いこうとか常用漢字じょうようかんじ以外いがいなどに
    猫が蝙蝠(こうもり)を食べるの?

  • CSSセレクタ: ルビ化かの範囲はんいを絞しぼり込こみ
    div.クラス>p

  • ルビの粒度りゅうど
    お気(き)に入(い)りorお気に入(きにい)りorお気に入り(おきにいり)

  • カタカナルビ
    アリスはそう呟(ツブヤ)いた。

  • その他た、括弧かっこの変更へんこうやrp要素ようその使用しようなど

インストール

なんとかしてgitとnpmコマンドの環境かんきょうを入いれたあと、こうする。

# 任意のディレクトリで、ダウンロード
git clone https://github.com/big-stream/rubygana.git

# インストール(依存ライブラリのダウンロード)
cd rubygana/
npm install --global

# もうrubyganaコマンドが使えるぞ
rubygana --help

依存いぞんライブラリに感謝かんしゃ

  • 日本語にほんご解析かいせき: kuromoji.js
  • マークダウンのタグ化か: marked
  • HTML解析かいせき: cheerio

使用しよう法ほう

rubygana [オプション...] [ファイル]

一ひとつのファイル指定していか標準ひょうじゅん入力にゅうりょくが必要ひつようで、結果けっかはすべて標準ひょうじゅん出力しゅつりょく。

echo '猫と蝙蝠' | rubygana

# 出力
猫(ねこ)と蝙蝠(こうもり)

ルビ化かの際さい、kuromoji.jsのビルドに1、2秒びょうかかります。

オプション

  1. HTMLのルビ化か: --html

    rubygana [-HKCy] [-g 学年] [-G 粒度]
         [-s セレクタ] [-n 除外セレクタ] [-N 除外要素]
         [-c [CSS]] [-T タイトル] [--ruby-size [フォント倍率]]
         [--ruby [フレーズ:]単語:ルビ]
         [--ruby-re 正規表現:単語:ルビ] [ファイル]

  2. テキスト文書ぶんしょのルビ化か: --text

    rubygana [-tKC] [-b 括弧] [-g 学年] [-G 粒度]
         [--ruby [フレーズ:]単語:ルビ]
         [--ruby-re 正規表現:単語:ルビ] [ファイル]

  3. 学年がくねんクラス名めい付加ふか: --add-class

    rubygana -a [-y] [クラス頭語] [-c [CSS]] [--switch]
         [-s セレクタ] [-n 除外セレクタ] [ファイル]

  4. マークダウンのHTML5化か: --md-html

    rubygana -m [-c [CSS]] [-T タイトル] [-L 見出し] [ファイル]

  5. テキスト文書ぶんしょのHTML5化か: --text-html

    rubygana -w [-c [CSS]] [-T タイトル] [-L 見出し] [ファイル]

その他たオプション(引数ひきすうなし): 

-A, --sample-text   サンプルテキスト出力  
-B, --sample-html   サンプルHTML出力  
-M, --sample-md     サンプルマークダウン出力
-d, --debug         デバッグ用  
-h, --help          ヘルプ  
--readme-md        詳細ヘルプ(このマークダウン)  
--readme-html    詳細ヘルプ(HTML)  
-v, --verbose       デバッグ情報を増やす  
-V, --version       バージョン情報  
-い, --依存          依存ライブラリ確認  
-ら, --ライセンス     ライセンス情報  
-☺ , --感謝

コマンドグループ:

--html, -H

HTML扱あつかいしてルビ化か。冒頭ぼうとうにDOCTYPEやxml宣言せんげんがあれば自動じどう判別はんべつのため、省略しょうりゃく可か。 

# DOCTYPEやxml宣言がないのにHTML扱いしたい場合
echo '蝙蝠なの?' | rubygana -H --only-body

# 出力
<ruby><rb>蝙蝠</rb><rt>こうもり</rt></ruby>なの?

--text, -t

テキスト扱あつかいしてルビ化か。

# DOCTYPEやxml宣言あるのにテキスト扱いしたい場合
echo '<!DOCTYPE html><html><body>蝙蝠なの?</body></html>' | rubygana -t

# 出力
<!DOCTYPE html><html><body>蝙蝠(こうもり)なの?</body></html>

--add-class, -a ['クラス頭語']

漢字かんじの学習がくしゅう学年がくねんをruby要素ようそのクラス名めいに付加ふか。すでにruby要素ようそがあるHTMLに対たいしてこの動作どうさだけを行おこない、新あらたなルビ化かはしない。学年がくねん別べつに色分いろわけしたいとか。
デフォルト: 学年

# 「落」は3年生、「下」は1年生なので
echo '落下' | rubygana -H | rubygana -a --only-body

# 出力
<ruby class="学年3 学年1"><rb>落下</rb><rt>らっか</rt></ruby>

# 中学生以降は「学年7」、常用漢字以外の漢字は「学年8」

--md-html, -m

ルビ化かはせず、単たんにマークダウンをHTML5文書ぶんしょに変換へんかん。h1要素ようそになるものをタイトル(title要素ようそ)にする。変更へんこうは--headlineと--title

--text-html, -w

ルビ化かはせず、単たんにテキスト文書ぶんしょをHTML5文書ぶんしょに変換へんかん。テキストは行ぎょうごとにp要素ようそに入いれ(空行くうぎょうは対象たいしょう外がい)、冒頭ぼうとうの行ぎょうを見出みいだし(h1要素ようそ)とタイトル(title要素ようそ)にする。変更へんこうは--headlineと--title

--sample-text, -A

練習れんしゅう用ようサンプルとして、ルビのない下記かき3行ぎょうテキストを標準ひょうじゅん出力しゅつりょく。

不思議の国のアリス
「猫が蝙蝠を食べるの? 蝙蝠が猫を食うの?」
アリスは白ウサギの穴に落下しながら呟いた。すると突然…

--sample-html, -B

練習れんしゅう用ようサンプルとして、ルビのないHTMLを標準ひょうじゅん出力しゅつりょく。

--sample-md, -M

練習れんしゅう用ようサンプルとして、ルビのないマークダウンを標準ひょうじゅん出力しゅつりょく。

ルビの調整ちょうせい:

--ruby '[フレーズ:]単語:ルビ'

ルビを指定していしたいなら。フレーズ(改行かいぎょうなし)で、フレーズ内ないの単語たんごのみに作用さよう。複数ふくすう回かい使用しよう可か。マッチした単語たんごには--gradeなど自動じどうルビ用ようオプションの影響えいきょうなし。フレーズに:が含ふくまれるなら代かわりに--ruby-commaを使用しよう。

# 自動のルビが不味いときや、工夫したいときに使う。
echo '反復横飛びの無限反復' | rubygana -r '無限反復:反復:ループ'

# 出力
反復(はんぷく)横(よこ)飛(と)びの無限(むげん)反復(ループ)


# --rubyはいくつも使用できるが、同じフレーズ・単語は先のものが優先
echo 猫と蝙蝠 | rubygana -r '猫:cat' -r '猫:きゃっと' -r '蝙蝠:bat'

# 出力
猫(cat)と蝙蝠(bat)


# --htmlでも
echo '猫の一撃をくらった' | rubygana --html -r '猫の一撃:ネコパーンチ' --only-body

# 出力
<ruby><rb>猫の一撃</rb><rt>ネコパーンチ</rt></ruby>をくらった

--ruby-comma '[フレーズ,]単語,ルビ'

--rubyの代かわりに使つかう。違ちがう点てんは引数ひきすうが,区切くぎり。

echo 'ねえ、:コマンドって知ってる?' | rubygana --ruby-comma ':,コロン'

# 出力
ねえ、:(コロン)コマンドって知(し)ってる?

--ruby-re '正規表現:単語:ルビ'

正規せいき表現ひょうげん(改行かいぎょうなし、:を使つかわないこと)を用もちいたルビ指定してい。マッチした表現ひょうげん中ちゅうの単語たんごのみに作用さよう。複数ふくすう回かい使用しよう可か。マッチした単語たんごには--gradeなど自動じどうルビ用ようオプションの影響えいきょうなし。正規せいき表現ひょうげんはJavaScriptのRegExpで評価ひょうか。

echo '1行2行、一行は四行も空行にした' | rubygana --ruby-re '[0-9四]行:行:ぎょう' --ruby '空行:くうぎょう'

# 出力
1行(ぎょう)2行(ぎょう)、一行(いっこう)は四(よん)行(ぎょう)も空行(くうぎょう)にした


# マッチした表現に単語が複数回現れる場合、一つ目にのみルビが付く
echo '激怒激怒' | rubygana --ruby-re '(激怒)+:激怒:プンプン'

# 出力
激怒(プンプン)激怒(げきど)


# 先読み表現なども使える
echo '激怒プンプン丸が激怒' | rubygana --ruby-re '激怒(?=プンプン丸):激怒:げきおこ'

# 出力
激怒(げきおこ)プンプン丸(まる)が激怒(げきど)

--grade, -g '学年'

この学年がくねんまでの漢字かんじはルビ化かしない。0-7を指定してい。2なら3年生ねんせい以降いこうの、6なら中学校ちゅうがっこう(常用漢字じょうようかんじの残のこり)以降いこうの、7で常用漢字じょうようかんじ以外いがいの漢字かんじをルビ化か。
デフォルト: 0

# 常用漢字以外だけルビ化なら7
rubygana --sample-text | rubygana -g 7

# 出力
不思議の国のアリス
「猫が蝙蝠(こうもり)を食べるの? 蝙蝠(こうもり)が猫を食うの?」
アリスは白ウサギの穴に落下しながら呟(つぶや)いた。すると突然…

学年がくねんの判断はんだんは、文科省もんかしょう学年がくねん別べつ漢字かんじ配当はいとう表ひょう、文化庁ぶんかちょう常用漢字じょうようかんじ表ひょうによる。

--granularity, -G '粒度'

かな交まじり単語たんごのルビの振ふり方かたの調整ちょうせい。0-2を指定してい。例たとえば「お気きに入いり」は1単語たんごであり、
0=> お気(き)に入(い)り
1=> お気に入(きにい)り
2=> お気に入り(おきにいり)
デフォルト: 0

echo 'お気に入り' | rubygana --html -G 2 --only-body

# 出力
<ruby><rb>お気に入り</rb><rt>おきにいり</rt></ruby>

--brackets, -b '括弧'

括弧かっこを変更へんこうしたいなら。例れい '[]'
デフォルト: ()

# 指定文字が奇数個なら、左側に一つ多く配分
echo '蝙蝠' | rubygana -b '(=)'

# 出力
蝙蝠(=こうもり)

--katakana, -K

カタカナのルビに(デフォルトはひらがな)

echo 'アリスはそう呟いた。すると突然…' | rubygana --katakana
=> アリスはそう呟(ツブヤ)いた。すると突然(トツゼン)…

HTMLの調整ちょうせい:

セレクタと除外じょがいセレクタはcheerioによる。

--only-body, -y

body要素ようその中身なかみだけ出力しゅつりょく(bodyタグ含ふくまず)(--commentは出力しゅつりょく)。以下いか例れいでも多用たよう。

--selector, -s 'セレクタ'

この要素ようその子孫しそんをルビ化か。これに該当がいとうした要素ようそ(のうち)そのものを--not-selectorなら除外じょがいできるが、--ng-elementsでは除外じょがいできない。
例れい: #id, p.クラス, ol>li:nth-child(1)
デフォルト: body

# ol要素直下の1番目li要素と、文クラスのp要素だけルビ化
rubygana --sample-html | rubygana -s 'ol>li:nth-child(1), p.文'


# 「要素>*」にすると、要素直下の漢字は対象にならない
echo '<p>猫と<span>蝙蝠</span></p>' | rubygana --html -s 'p>*' --only-body

# 出力
<p>猫と<span><ruby><rb>蝙蝠</rb><rt>こうもり</rt></ruby></span></p>

--not-selector, -n '除外セレクタ'

-sそのものから除外じょがい(子孫しそんを除外じょがいするのではない)。-sのデフォルトはbodyなので、-sと一緒いっしょに使つかう。
例れい: li.クラス
デフォルト(-hの時とき): ruby,script,style,code,pre,samp,blockquote
デフォルト(-aの時とき): 空から

# p要素をルビ化したいが、そのうち無視クラスを除外したいとき
echo '<p class="無視">猫</p><p>蝙蝠</p>' | rubygana --html -s 'p' -n '.無視' --only-body

# 出力
<p class="無視">猫</p><p><ruby><rb>蝙蝠</rb><rt>こうもり</rt></ruby></p>


# 該当した`-s`そのもの(と子孫)を除外できるが、該当しなかった`-s`の子孫は除外できない
echo '<div><p>猫</p></div><p>蝙蝠</p>' | rubygana --html -s 'body>*' -n 'p' --only-body

# 出力
<div><p><ruby><rb>猫</rb><rt>ねこ</rt></ruby></p></div><p>蝙蝠</p>

--ng-elements, -N '除外要素'

-sの子孫しそんから除外じょがいする要素ようそを|区切くぎりで。-sそのものから除外じょがいするには、--not-selector
デフォルト: ruby|script|style|code|pre|samp|blockquote

# 絶対に除外したい要素で使う。(最低限scriptとstyleは除外推奨)
echo '<p>猫と<a>蝙蝠</a></p>' | rubygana --html -N 'a' --only-body

# 出力
<p><ruby><rb>猫</rb><rt>ねこ</rt></ruby>と<a>蝙蝠</a></p>

--css, -c ['CSS']

head要素ようそ末尾まつびにstyle要素ようそ追加ついか。そのCSSを指定してい。引数ひきすう省略しょうりゃくで下記かきデフォルトCSS。

  • --htmlのとき、ルビのフォントサイズ(漢字かんじに対たいする倍率ばいりつ)を0.5(又または--ruby-size)倍ばいに。
    • ruby>rp,ruby>rt{font-size:0.5em;}
  • --add-classのとき、常用漢字じょうようかんじ以外いがいの漢字かんじのルビの色いろをDarkRedに。
    • ruby[class$="8"]>rt{color:DarkRed;}
  • --text-htmlと--md-htmlのとき、空から(のstyle要素ようそ)

--ruby-size [フォント倍率]

head要素ようそ末尾まつびに下記かきstyle要素ようそ追加ついか。ルビのフォントサイズ(漢字かんじに対たいする倍率ばいりつ): 0.1-2.0(デフォルト0.5)

  • ruby>rp,ruby>rt{font-size:倍率em;}

--use-rp

rp要素ようそ(ルビ非ひ対応たいおうブラウザのための括弧かっこ書がき)を使用しようする。

echo 猫なの? | rubygana -H --use-rp --only-body

 # 出力
<ruby><rb>猫</rb><rp>(</rp><rt>ねこ</rt><rp>)</rp></ruby>なの?

--title, -T 'タイトル'

title要素ようその内容ないよう。

--headline, -L '見出し'

h1要素ようその内容ないよう。--text-htmlと--md-htmlで使つかう。

--switch

ページトップにルビの学年がくねんスイッチを追加ついか。漢字かんじの学習がくしゅう学年がくねんごとにルビをオン・オフ可能かのう。

その他た:

--comment, -C

rubyganaコマンド実行じっこうに関かんするコメントを末尾まつびに追加ついか。

--htmlのとき: body要素ようそ末尾まつびに改行かいぎょうと下記かき4行ぎょう挿入そうにゅう。スタイルがdisplay:noneのため、ソースからなら確認かくにんできるコメント。

<!-- この文書のルビ振りは下記コマンド(rubygana VERSION)を用いました。 -->
<pre style="display:none;"><code>
ここに実際に実行したコマンドオプション・引数が入る
</code></pre>

--textのとき: 文書ぶんしょ末尾まつびに改行かいぎょうと下記かき2行ぎょう追記ついき。

# この文書のルビ振りは下記コマンド(rubygana VERSION)を用いました。
# ここに実際に実行したコマンドオプション・引数が入る

--debug, -d

開発かいはつ用ようオプション。kuromojiによる解析かいせき詳細しょうさい(形態素けいたいそ)などが確認かくにんできる。-ddでオプション引数ひきすうの確認かくにんだけして終了しゅうりょう。

--verbose, -v

開発かいはつ用ようオプション。debug情報じょうほうを増ふやす。

1行ぎょうルビ族ぞく

# ダウンロードし、ルビ化し、firefoxで開く
page='ja.wikipedia.org/wiki/不思議の国のアリス'; wget --page-requisites --convert-links "https://$page" && rubygana $page > $page.ruby.html; firefox $page.ruby.html &


# 入力の文字コードはUTF-8のみ受付:
# 違うならiconvコマンドなどで変換してから
cat shift-jis.txt | iconv -f 'shift-jis' -t 'utf8' | rubygana
# (標準出力の文字コードは端末の設定で変更)


# サンプルテキストを出力し、それをHTML化し、次にh1だけカタカナでルビ振りし、そしてp要素を2年生超でルビ振りし、
# そして学年クラス名を付加して中学生以上常用漢字ルビは青色で常用漢字以外ルビは赤色スタイルにして、
# それをalice.htmlとして保存し、それをFirefoxで開く。
rubygana -A | rubygana -w | rubygana -HKs 'h1' | rubygana -s 'p' -g 2 | rubygana -ac 'ruby[class$="7"]>rt{color:blue;}ruby[class$="8"]>rt{color:red;}' > alice.html; firefox alice.html &

rubygana --readme-html
